## Usage

### Step 1: Run the Main Pipeline

```bash
python sheffield_ml_main.py
```

This executes:
- Data loading and sanity checks
- Preprocessing and cleaning
- Feature engineering
- All classification tasks (multiclass, binary, categorical)
- Regression tasks and trend analysis
- Unsupervised clustering (KMeans, DBSCAN, Agglomerative)
- Performance evaluation and visualisations
- Saves cleaned CSV: `Sheffield Collision Data Cleaned.csv`

### Step 2: Run the Improvements File

```bash
python sheffield_ml_improvements.py
```

This adds:
- Improved IQR-based outlier capping (fixes the mean-only imputation)
- 4 additional classification tasks (road surface, light conditions, police attendance, weekend)
- ROC-AUC curves for all binary/multiclass tasks
- 3 additional regression targets (vehicles, spatial risk, polynomial trend)
- 3 additional clustering tasks (geographic, temporal, hierarchical)
- Folium interactive heatmap → `sheffield_collision_heatmap.html`
- PCA as a preprocessing step (not just visualisation)
- Comprehensive performance evaluation table
- Residual analysis for regression
- Responsible AI documentation

### Step 3: View the Heatmap

Open `sheffield_collision_heatmap.html` in any web browser.
Interactive features: zoom, pan, toggle layers, click markers for details.

### Step 4: Run the Dashboard

```bash
python dashboard/dashboard.py
```

Navigate to `http://127.0.0.1:8050` in your browser.
